# CPEN 221 – Software Construction I

## Purpose of the course

CPEN 221 develops reliable software through abstraction, specifications, data types, testing, object-oriented design, functional operations, and concurrency. This page is a conceptual map, not a reconstruction of the official syllabus.

## Main conceptual units

Software Abstraction and Specifications; Abstract Data Types; Representation Invariants and Abstraction Functions; Testing and Verification in Software; Object-Oriented Design; Java Streams; Optional in Java; Reduction Operations; Imperative vs Functional Java; Concurrent Software Design; Git Remotes and SSH.
