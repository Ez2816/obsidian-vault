## Future source-ingestion rule

When course materials are uploaded into 10 Sources/:

1. identify the relevant course;
2. identify related seed pages;
3. compare the original material with the seed content;
4. preserve useful intuitive explanations;
5. correct inaccurate or unsupported claims;
6. add course-specific terminology;
7. add equations with variable definitions;
8. add diagrams or figure references when useful;
9. distinguish lecture content from Elsa's own interpretation;
10. cite the exact Source path;
11. update source_status;
12. update confidence;
13. record contradictions;
14. update this build tracker;
15. update the main index when appropriate;
16. append an ingestion entry to the system log;
17. report every file changed.

Do not delete an intuitive explanation only because a course source uses more formal wording. Improve the page so it contains both.
